Movie Collection App

Project on Github

Description: basic app written on Python to store the favourite movies into a collection with the possibility to add, view or delete a movie.

Input commands:

  • "info", "i" - readme information
  • "add", "a" - add a movie to collection
  • "display", "d" - display movie collection
  • "delete", "del" - delete a movie
  • "quit", "q", "end", "exit" - quit the program

Open from 'Command Prompt (cmd)':

python movie_app.py

"To Do" app (simplistic version)

Project on Github

'TO DO' app written on Python (simplistic version)

  • Name: "TO DO" app (simplified TXT version)
  • Version: 0.1 (beta)
  • Author: Valeriy B.

The app works from keyboard inputs:

  • Type a task you want to add to the 'TO DO' app list;
  • Type '!print' command to print the tasks from the app list;
  • Type '!remove' command to remove a task from the list;
  • Type '!finish' to stop the app.

When '!finish' command is typed, a 'tasks.txt' file is created saving the latest data entered before command input.
